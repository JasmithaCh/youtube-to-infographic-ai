# YouTube to Infographic AI Generator

## Overview
This project is an n8n workflow that takes a YouTube video, extracts the transcript, analyzes the content with AI, and generates an infographic-style image from the main points.

## Workflow
1. User enters a YouTube link
2. Transcript is fetched
3. AI extracts the core idea and key points
4. A prompt is generated from the points
5. Stability AI generates the final image

## Tools Used
- n8n
- Apify / transcript source
- AI prompt generation
- Stability AI image generation

## Files
- `workflow.json` — exported n8n workflow
